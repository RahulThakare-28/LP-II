## Problem Statement (chit-1)
Create a responsive web page that shows an admin dashboard for the Exam management system with a sidebar, table, and statistics in cards using HTML, CSS, and Bootstrap


## Exam Management System - Admin Dashboard
## 🎯 Aim
To create a responsive Admin Dashboard for an Exam Management System using HTML, CSS, and Bootstrap.

---

## 🧰 Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (via CDN)

---

## 🌐 Bootstrap CDN
Bootstrap is used for responsive design and UI components.

## 🏗️ Structure of the Web Page

The page is divided into three main sections:

### 1. Sidebar (Navigation Panel)
- Located on the left side
- Contains links:
  - Dashboard
  - Students
  - Exams
  - Results
- Styled with dark background

---

### 2. Statistics Cards
- Display key information:
  - Total Students
  - Exams Conducted
  - Results Published
- Implemented using Bootstrap cards

---

### 3. Data Table
- Displays recent exam results
- Columns:
  - ID
  - Student Name
  - Exam
  - Marks
- Styled using Bootstrap table classes

---

## 📐 Layout Design (Bootstrap Grid)

container-fluid  
└── row  
  ├── col-md-2 → Sidebar  
  └── col-md-10 → Main Content
